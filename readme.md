## Installation

```sh
$ apm install imagemin
```

## Usage

- Packages > Imagemin > **Minify image**
- Context Menu > **Minify image**
- Command Palette (<kbd>⌘</kbd> + <kbd>shift</kbd> + <kbd>P</kbd>) > **Minify image**
